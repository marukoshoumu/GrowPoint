# グローポイント AI支援記録自動化

Google Apps Script（GAS）で、面談音声から記録票・モニタリング関連ドキュメントまでを自動生成するためのソースです。

## リポジトリの方針

このリポジトリは **ソースコード（主に `gas/`）を中心**に管理します。設計メモ・仕様ドラフト・音声サンプルなど、作業用の資料は **`docs/` 配下に置き、Git には上げない**運用を想定しています（`.gitignore` で `docs/` を除外）。

ローカルにだけあるファイル（テンプレ作成用の指示文など）を参照する手順は、[SETUP.md](SETUP.md) に記載しています。

## ディレクトリ構成

| パス | 内容 |
|------|------|
| `gas/` | Apps Script の `.gs` と `appsscript.json`（エディタへコピーまたは clasp で連携） |
| `SETUP.md` | スプレッドシート作成から Script Properties、運用・トラブルシュートまでの手順 |
| `docs/` | （任意・ローカル）設計書・仕様・テンプレ指示など。リポジトリクローンには含まれない想定 |

## はじめ方

詳細は **[SETUP.md](SETUP.md)** を参照してください。

1. Google スプレッドシートと GAS プロジェクトを用意する  
2. `gas/` の各ファイルをプロジェクトに反映する  
3. Script Properties と初回セットアップを行う
